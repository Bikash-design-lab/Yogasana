# Yogasana

The app should encourage users to engage in wellness activities (Yogasanas) by making it fun and competitive.

Command run on terminal:
Backend-

1. npm init -y
2. npm i express
3. npm i nodemon
4. npm i mongoose
5. npm i cors
6. npm i morgan
7. npm i dotenv

Frontend-

1. npm create vite@latest
2. Proj name - Frontend
3. cd Frontend
4. npm i
5. npm i react-router-dom
